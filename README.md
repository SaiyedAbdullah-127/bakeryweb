## Supabase Configuration

To ensure a smooth login experience during development, you may want to disable email verification:

1. Go to your [Supabase Dashboard](https://supabase.com/dashboard).
2. Select your project.
3. Go to **Authentication > Providers > Email**.
4. Toggle **Confirm email** to **OFF**.
5. Click **Save**.

This allows you to sign up and log in immediately without waiting for a confirmation email.

First, run the development server:

```bash

# or

# or
```
